Usage:


Run the executable "Alignment.exe". It will open and Window application with a dialog box, and a console window. Ignore the console window. (But don't close it)

In the dialog box, fill the form:

Subject String: a string of upper-case alphabetic (no delimiters).

Pattern String: a string of upper-case alphabetic (no delimiters).

k-tup:	an integer between 2 to 6, inclusively.

Button "Report": generate a report to an HTML file named "CS599_report.html" in the same directory of the .exe file.

Button "Exit": quit the application
